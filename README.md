# madlibs
Following the tutorial of https://auth0.com/blog/rxjs-advanced-tutorial-with-angular-web-speech-part-1/

# Setup
## Backend
```
$ git clone https://github.com/kmaida/madlibs-api
$ cd madlibs-api/
$ yarn
$ cd ../
```

## Frontend
```
$ cd madlibs/
$ yarn
$ cd ../
```

# Run
## Backend
```
$ cd madlibs-api/
$ node server
```

## Frontend
```
$ cd madlibs/
$ yarn start
```
